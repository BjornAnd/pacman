Pac-Man source code

There are a few ways to get the source code.

Git

To get the full repository, install git and from a terminal run the following: 

$ git clone http://pacman.shaunew.com/.git pacman

When prompted for username/password, input the same information you used to log in to this website. This is read-only access.

Download

You can download the current version in a zip file.

Browse

You can browse the code here.
